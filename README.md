# 🧠 Digital Diet and Mental Health Prediction

> An AI-powered Machine Learning project that predicts an individual's mental health status based on digital lifestyle habits such as screen time, social media usage, sleep, and physical activity.

---

## 📌 Project Overview

The increasing use of digital devices has significantly impacted mental well-being. This project analyzes digital lifestyle patterns and predicts mental health outcomes using Machine Learning algorithms.

The goal is to identify how digital habits influence mental health and build an intelligent prediction model.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Zeel2356/Digital-Diet-and-Mental-Health-Prediction.git
```

Move into the project directory

```bash
cd Digital-Diet-and-Mental-Health-Prediction
```

Install dependencies

```bash
pip install pandas numpy matplotlib scikit-learn notebook
```

Launch Jupyter Notebook

```bash
jupyter notebook
```

Open

```
project final.ipynb
```
